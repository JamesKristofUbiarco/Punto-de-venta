# Punto de venta en C++
Un proyecto de la carrera de desarrollo de software en el **CETI Colomos** y el curso de git y github en **Platzi**, en el proyecto de C++ se tienen que leer varios códigos de barras y escribir el producto que se añade.
* Si pulsas 0 en el lector de códigos de barras te imprime un ticket de compra con los productos que hayas agregado.
* Si pulsas -1 en el lector de códigos de barras te muestra el flujo de caja del dia: productos vendidos e ingresos totales.
* Despues de escribir el código de barras de algún producto el programa te va a decir "Pulsa cualquier caracter y pulsa enter para continuar", solo es cuestión de seguir la instrucción y el programa te permitirá agregar mas productos a la venta
